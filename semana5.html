Semana 5

CLASE PRESENCIAL:

Para esta clase el profesor había solicitado que todos traigan sus laptops para poder avanzar la bitácora en clase y ayudar a todo aquel que aun presente problemas con la codificación de la página web.

Como prueba realizamos una lluvia de ideas de todo lo aprendido durante estas 4 semanas tanto de manera presencial como virtual.

CLASE VIRTUAL:

En esta clase creamos páginas web, comenzamos creando una página web en Google Sites, conocimos los usos de sus herramientas y funciones. También creamos una página web en Canva, conocimos las distintas funciones y como podríamos usar diferentes plantillas predeteminadas y modificarlas a nuestro gusto.

Luego vimos otras páginas con las cuáles tan sólo ralizando un comando y gracias a la Inteligencia Artificial (AI) podiamos crear la página, siendo: Notion, Framer, Hocoos, Wix, Wordpress

Por último el profesor nos mando a salas para definir que programa de todos los anteriores mencionados utilizariamos para la creación de nuestra página web para el proyecto.
